从USB安装ubuntu

二 27 八月 2013
By zxkletters

In tech.

今天通过usb安装ubuntu系统时遇到的小问题，记录下。

按照官网的文档 制做好usb stick(通过Universal-USB-Installer-1.9.3.9.exe 软件，将ubuntu iso文件flush到usb)，将usb插入电脑并在bios里设置通过usb启动后，电脑一直处于黑屏状态。 后来在ubuntu社区查找解决办法 发现了一个帖子Installing Ubuntu with USB stick, 提到这个软件Rufu可以解决. 最后按照该贴用Rufu制作的usb stick成功安装。
